Processo Seletivo
Projeto em Java que simula um processo seletivo de candidatos. O sistema aplica conceitos de lógica de programação e Programação Orientada a Objetos (POO), incluindo:
Seleção de candidatos baseada no salário pretendido,
Contato com candidatos simulando tentativas de ligação,
Impressão de candidatos selecionados com índice,
Análise e decisão automatizada sobre a seleção,
Este projeto serve como exercício de POO, loops, arrays, métodos e boas práticas de código em Java.

Funcionalidades
Seleção de candidatos: seleciona até 5 candidatos com salário compatível,
Contato com candidatos: simula tentativas de ligar para o candidato, com até 3 tentativas,
Impressão de candidatos: imprime a lista de candidatos com índice e usando for-each,
Análise de candidatos: determina se deve ligar para o candidato, oferecer contra proposta ou aguardar resultados.

Tecnologias
Java 17 (ou superior),
Conceitos de Programação Orientada a Objetos (POO),
Estruturas de repetição (for, while),
Arrays e métodos estáticos
